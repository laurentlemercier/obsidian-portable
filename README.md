# obsidian-portable
Portable version of Obsidian : no installation needed

# how it was created
cf https://share.note.sx/3vhfcdlc, using a fresh installation of obsidian (https://obsidian.md/download)

# How to use
Download either .7z or .zip, uncompress, put the PAF exe file in your favorite folder (USB supported).
Just execute the PAF exe file, then open your favorite vault (or create it)
